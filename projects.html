1. Pizza Ordering App

Academic Project
Java, JavaFX, JDBC & SQL Database

The project uses a system to help pizza stores manage their inventory, orders and customer data. Customer can customize their pizza by choosing different ingridents, which are tracked in the real time to keep stock levels sufficient. 2. Banking System

Academic Project

Java, JavaFX & IntelliJ IDEA

The project is a JavaFX based banking application. It supports transactions like deposits, withdrawals, and transfers, updating account balances in real time. The system has strong security to ensure only authorized users can access it. It features a user-friendly interface, improving the overall experience. The system allows users to create and customize account type, with specific interest rates and transaction limits.

3. Interactive 3D Carousel

Personal Project

HTML, CSS & JavaScript

The project is a 3D carousel display, which features a visually engaging stact of image cards. When activated, these images flip open and arrange themselves in a distinctive formation. As the display continues, the images rotate in place, creating a dynamic interactive experience. The movement of the images is also responsive to the mouse movements, adjusting the distance between the image cards based on cursor position.
